```

```

## issue
追加したい機能、修正したいバグなどはissueに書いていく。作成したブランチとissueを紐づけておく

## ブランチ
- *main* → マージしたらデプロイが走る
- *develop* → 開発用

## デバッグの流れ
1. *develop*ブランチから新しいブランチを作成する
2. 変更をcommmitしたらプルリクを作成する
3. マージして完了
